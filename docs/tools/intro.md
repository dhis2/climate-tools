# Tools

DHIS2 Climate Tools brings together a set of Python-based geospatial and climate analysis libraries that are commonly used throughout the guides and workflows in this toolkit.

In this section, we provide a brief overview of the main tools and technologies included in the DHIS2 Climate Tools environment, along with pointers to where and how they are used in practice. The goal is not to document these tools in full, but to help you understand their role in typical climate and health workflows.

- [dhis2eo](dhis2eo.md)
- [dhis2-python-client](dhis2-python-client.md)
- [xarray](xarray.md)
- [geopandas](geopandas.md)
- [earthkit](earthkit.md)
